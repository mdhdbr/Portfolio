ResConnect - Aviation PNR Management

ResConnect is a specialized system developed for managing Passenger Name Records (PNRs) and booking modifications in the aviation industry. This solution streamlines passenger request processing while maintaining data integrity and audit trails.

Core Features

  • EK -airline booking reference management
